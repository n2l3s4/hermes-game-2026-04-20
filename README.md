# Bobo's Ocean Match

A fun sea creature memory matching game for kids! 🐠🐙🐳

Flip the cards to find matching pairs of ocean friends. How fast can you match them all?

## Play the Game

**🎮 [Play Now](https://n2l3s4.github.io/hermes-game-2026-04-20/)**

## How to Play

- Click or tap a card to flip it over.
- Find two cards with the same sea creature to make a match.
- Match all the pairs to win!
- Listen for the happy sounds when you get it right.
